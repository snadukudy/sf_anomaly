# Manufacturing Sensor Anomaly Tracker

## Overview
The Manufacturing Sensor Anomaly Tracker project aims to monitor and analyze sensor data from manufacturing systems to identify and respond to anomalies. This tool leverages advanced algorithms and machine learning techniques to ensure operational efficiency and reduce downtime.

## Features
- Monitoring of sensor data.
- Anomaly detection using Snowflake Cortex ML Anomaly function
- Statistical Process Control Analysis of sensor data
- Dashboard for visualizing the sensor data and anomalies using Streamlit.

## Getting Started in Minutes
1. Use the repository: `git clone https://github.com/snadukudy/sf_anomaly` as a reference to build the anomaly model, statistical process control and streamlit visualization in Snowflake data platform.
