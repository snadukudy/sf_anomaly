# Manufacturing Sensor Anomaly Tracker

## Overview
The Manufacturing Sensor Anomaly Tracker project aims to monitor and analyze sensor data from manufacturing systems to identify and respond to anomalies. This tool leverages advanced algorithms and machine learning techniques to ensure operational efficiency and reduce downtime.

## Features
- Real-time monitoring of sensor data.
- Anomaly detection using machine learning models.
- Dashboard for visualizing sensor data and anomalies.
- Alerts and notifications for immediate action.

## Getting Started
1. Clone the repository: `git clone https://github.com/snadukudy/sf_anomaly`
2. Install required dependencies: `pip install -r requirements.txt`
3. Run the application: `python app.py`
